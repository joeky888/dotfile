### 藥名

- 化學名：正式學術名稱，例如：1-(isopropylamino)-3-(1-naphthyloxy)propan-2-ol
- 一般名：全球通用簡稱，例如：普潘奈 (propranolol)
- 商品名：製藥公司自創名字，例如，”生達“心率錠(生達化學製藥)、”健心寧(人人化學製藥)“

### 一、憂鬱 (Anti-depressants)

四大抗憂藥物

- 三環抗鬱劑（TCA）
  - 最早出現的抗憂鬱劑，因副作用多現為二線用藥
  - 常見副作用：心悸、嗜睡、口乾、便秘、姿勢性低血壓
  - 禁忌：青光眼、前列腺肥大、心臟傳導異常患者慎用
- 選擇性血清素回收抑制劑（SSRIs）
  - 第一線用藥，副作用較TCA少，但可能影響性功能
  - 常見副作用：噁心、失眠、頭痛、性慾減退（高達70%患者）
  - 優點：過量安全性高，適合長期使用
- 正腎上腺素與血清素回收抑制劑（SNRI）
  - 雙重作用機轉，對伴隨疼痛的憂鬱症效果較佳
  - 常見副作用：血壓輕微升高、出汗、焦躁感
  - 特殊用途：亦用於廣泛性焦慮症、纖維肌痛症
- 其他
  - MAOI（如沙夫肼）：需嚴格限制飲食（避免酪胺酸食物）
  - NaSSA（如米氮平）：顯著增加食慾與體重
  - 其他如美妥平、威克倦、美舒鬱、煩多閃

### 二、躁鬱、癲癇 (Mood Stabilizers, AEDs)

分成抗躁鬱與抗癲癇，兩種藥皆可使用

抗躁鬱劑 (Mood Stabilizers)

- 鋰鹽 (Lithium)
  - 治療躁鬱症的一線用藥
  - 副作用包括手抖、口渴、多尿、體重增加、甲狀腺功能異常
  - 需定期監測血中濃度（治療窗狹窄）
- 抗癲癇藥物轉用
  - 部分抗癲癇藥物具情緒穩定作用，常用於躁症治療
  - 副作用較鋰鹽少，但可能嗜睡、頭暈、腸胃不適

抗癲癇藥物 (AEDs)

- 傳統抗癲癇藥
  - 作用機轉多為抑制神經元過度放電
  - 常見副作用：嗜睡、頭暈、複視、認知功能影響
- 新一代抗癲癇藥
  - 副作用較少，但可能仍有體重變化、皮疹等問題
  - 部分兼具抗躁鬱效果

- 苯妥英：長期使用可能導致牙齦增生、骨質疏鬆

### 三、抗焦慮、失眠 (Anxiolytics & Hypnotics)

- 苯二氮平類 (Benzodiazepines, BZDs)
  - 作用：增強GABA抑制效果，快速緩解焦慮、誘導睡眠
  - 副作用：嗜睡、頭暈、記憶減退、依賴性（長期使用）
  - 禁忌：避免與酒精併用，呼吸抑制風險
- 非苯二氮平類 (Non-BZDs, Z-drugs)
  - 作用：選擇性作用於GABA受體，專注於改善失眠
  - 副作用：較少依賴性，但可能夢遊、頭痛
  - 優點：半衰期短，適合入睡困難者
- 抗憂鬱劑 (輔助治療焦慮)
  - SSRIs/SNRIs：長期焦慮症首選（如廣泛性焦慮症、恐慌症）
  - 低劑量TCA：用於夜間鎮靜（如阿米替林）
- 其他
  - 褪黑激素受體促效劑（如Ramelteon）：調節生理睡眠
  - 抗組織胺（如Diphenhydramine）：短期助眠，但易耐受

臨床注意事項

- BZDs 使用原則
  - 短期使用（<4週），避免長期依賴
  - 老年人需減量（跌倒風險↑）
  - 突然停藥可能引發反彈性失眠/焦慮
- Z-drugs 安全性
  - 唑吡坦可能導致夢遊（避免睡前活動）
  - 肝功能異常者需調整劑量
- 抗憂鬱劑轉換
  - SSRIs（如帕羅西汀）初期可能加重焦慮，需低劑量起始
- 非藥物建議
  - 認知行為治療（CBT-I）為慢性失眠首選
  - 睡眠衛生（固定作息、避免咖啡因）

### 四、心臟病、高血壓 (Cardiovascular & Antihypertensives)

藥物分類與特性

- 降血壓藥物 (Antihypertensives)
  - ACE抑制劑 (ACEIs)：抑制血管收縮，保護腎功能（但可能引起乾咳）
  - ARBs (血管張力素受體阻斷劑)：類似ACEIs，但較少咳嗽副作用
  - 鈣離子阻斷劑 (CCBs)：分為二氫吡啶類（降壓為主）與非二氫吡啶類（降心率）
  - 利尿劑 (Diuretics)：減少體液滯留，適用於鹽敏感型高血壓
  - β-受體阻斷劑 (Beta-blockers)：降低心臟負荷，但可能影響血糖代謝
- 抗心律不整藥物 (Antiarrhythmics)
  - Class I (鈉離子阻斷劑)：如Lidocaine（心室心律不整）
  - Class II (Beta-blockers)：如Metoprolol（心房顫動）
  - Class III (鉀離子阻斷劑)：如Amiodarone（廣效，但副作用多）
  - Class IV (鈣離子阻斷劑)：如Verapamil（PSVT）
- 抗心絞痛藥物 (Antianginals)
  - 硝酸鹽類 (Nitrates)：擴張冠狀動脈，緩解急性胸痛
  - Ranolazine：抑制心肌缺氧，減少心絞痛發作
- 抗凝血/抗血小板藥物
  - Warfarin (維生素K拮抗劑)：需監測INR值
  - DOACs (新型口服抗凝血劑)：如Apixaban、Rivaroxaban
  - Aspirin (抗血小板)：預防心肌梗塞

臨床注意事項

- 高血壓治療原則
  - 首選單一藥物：通常從ACEIs/ARBs或CCBs開始
  - 聯合治療：若單一藥物無效，可組合ACEI+利尿劑或CCB+Beta-blocker
  - 禁忌症：
    - Beta-blockers：氣喘、嚴重心搏過緩
    - ACEIs/ARBs：妊娠、雙側腎動脈狹窄
- 抗心律不整藥物風險
  - Amiodarone：長期使用可能導致肺纖維化、甲狀腺功能異常
  - Class IC藥物（如Flecainide）：禁用於結構性心臟病（增加猝死風險）
- 抗凝血劑監測
  - Warfarin：維持INR 2-3（機械瓣膜患者需更高）
  - DOACs：腎功能不全需調整劑量（如Apixaban）
- 心絞痛急性發作
  - 舌下NTG：若5分鐘未緩解，可重複一次（最多3次）
  - 禁忌：避免與PDE5抑制劑（如威而鋼）併用（嚴重低血壓）

### 五、糖尿病、痛風 (Cardiovascular & Antihypertensives)

藥物分類與特性

1. 糖尿病藥物 (Antidiabetics)

- 胰島素 (Insulin)
  - 速效型（如Lispro）：餐前注射，控制飯後血糖
  - 長效型（如Glargine）：基礎胰島素，穩定空腹血糖
  - 副作用：低血糖、體重增加、注射部位脂肪增生
- 口服降血糖藥 (OADs)
  - 雙胍類 (Biguanides)：如Metformin，第一線用藥，改善胰島素阻抗
  - 磺醯脲類 (Sulfonylureas)：刺激胰島素分泌，但低血糖風險高
  - SGLT2抑制劑：促進尿糖排出，兼具心腎保護
  - GLP-1受體促效劑：注射劑，減重效果顯著

2. 痛風藥物 (Gout Medications)

- 急性發作治療
  - NSAIDs（如Indomethacin）：快速消炎止痛
  - 秋水仙素 (Colchicine)：抑制白血球趨化，但腸胃副作用強
  - 類固醇（如Prednisolone）：無法耐受NSAIDs時使用
- 長期降尿酸治療
  - Allopurinol：抑制尿酸生成，需注意過敏反應
  - Febuxostat：適用於Allopurinol不耐受者，心血管風險爭議
  - Probenecid：促進尿酸排泄，腎結石患者慎用

糖尿病治療重點

- Metformin：
  - 禁忌：eGFR <30避免使用，顯影劑檢查前暫停
  - 優點：不引起低血糖，可能延緩糖尿病進展
- 胰島素使用技巧：
  - 輪替注射部位（腹部、大腿、手臂）避免脂肪增生
  - 低血糖處理：立即補充15g葡萄糖（如糖水、果汁）
- 新型藥物選擇：
  - SGLT2抑制劑：eGFR <45需減量，注意泌尿道感染
  - GLP-1促效劑：常見噁心（隨治療時間減輕）

痛風治療重點

- 急性期：
  - 避免降尿酸藥：可能延長發作，待緩解後2週開始使用
  - Colchicine：早期使用效果佳（發作12小時內）
- 長期降尿酸：
  - 目標值：血清尿酸<6 mg/dL（嚴重者<5 mg/dL）
  - Allopurinol：從低劑量起始（100 mg/日），緩慢增量
- 生活型態調整：
  - 限制高嘌呤食物（內臟、海鮮、酒精）
  - 充足水分（每日>2000 mL）減少結石風險

### 六、呼吸系統與過敏藥物 (Respiratory & Allergy Medications)

藥物分類與特性

1. 氣喘與慢性阻塞性肺病 (COPD) 藥物

- 支氣管擴張劑
  - 短效β2促效劑 (SABA)：如Salbutamol，急性發作急救用
  - 長效β2促效劑 (LABA)：如Salmeterol，需搭配類固醇
  - 抗膽鹼藥物 (LAMA)：如Tiotropium，COPD首選
- 抗發炎藥物
  - 吸入型類固醇 (ICS)：如Fluticasone，長期控制氣喘
  - 口服類固醇：如Prednisolone，急性惡化時短期使用

2. 過敏性鼻炎與蕁麻疹藥物

- 抗組織胺
  - 第一代：如Diphenhydramine，嗜睡副作用強
  - 第二代：如Cetirizine，較少中樞神經影響
- 鼻用類固醇：如Mometasone，改善鼻塞與打噴嚏

3. 止咳與祛痰藥物

- 中樞性止咳藥：如Dextromethorphan，抑制咳嗽中樞
- 化痰藥：如Acetylcysteine，分解黏液

臨床注意事項

氣喘與COPD治療

- 階梯式治療：
  - 輕度氣喘：SABA必要時使用
  - 中重度：ICS+LABA維持治療
- 吸入器技巧：
  - 使用後需漱口（避免口腔念珠菌感染）
  - 儲霧艙輔助兒童與老年人使用
- 急性惡化處理：
  - SABA：每20分鐘重複一次，最多3劑
  - 口服類固醇：嚴重時使用5-7天

過敏治療重點：

- 抗組織胺選擇：
  - 第一代：夜間瘙癢嚴重時使用（如Diphenhydramine）
    - 適用：夜間瘙癢、過敏急性期需快速止癢
    - 副作用：嗜睡、口乾、尿滯留（老年人慎用）
    - 禁忌：青光眼、前列腺肥大、駕駛前避免
  - 第二代：優先選擇（如Cetirizine、Loratadine）
    - 適用：長期過敏控制（如鼻炎、慢性蕁麻疹）
    - 優勢：不影響認知功能，兒童與老年人更安全
- 鼻用類固醇：
  - 需持續使用（效果2-3天後顯現）
  - 噴頭方向避免鼻中隔

止咳與祛痰藥物：

- 止咳藥：
  - 避免用於多痰咳嗽
  - Dextromethorphan過量可能致幻
- 化痰藥：
  - Acetylcysteine可霧化吸入（COPD急性期）

### 七、消化系統藥物 (Gastrointestinal Medications)

藥物分類與特性

1. 胃酸相關疾病藥物

- 質子幫浦抑制劑 (PPIs)：強效抑制胃酸，治療胃潰瘍、胃食道逆流
- H2受體拮抗劑 (H2 blockers)：中度抑制胃酸，適合輕症或維持治療
- 制酸劑 (Antacids)：快速中和胃酸，緩解偶發性不適

2. 腸胃蠕動與止吐藥物

- 促腸胃蠕動藥：如Metoclopramide，改善胃輕癱
- 止吐藥：如Ondansetron，抑制化療或術後嘔吐

3. 瀉劑與止瀉藥物

- 瀉劑：分為膨脹性、滲透性、刺激性
- 止瀉藥：如Loperamide，抑制腸道過度蠕動

4. 肝膽疾病藥物

- 利膽劑：如Ursodeoxycholic acid，改善膽汁淤積
- 肝臟保護劑：如Silymarin，輔助治療肝炎

臨床注意事項

1. 胃酸抑制劑使用

- PPIs：
- 長期使用可能增加骨折、低鎂血症風險
- 建議最低有效劑量，定期評估停藥可能
- H2 blockers：
- Cimetidine抑制CYP450，影響華法林、茶鹼代謝

2. 止吐藥選擇

- 化療嘔吐：5-HT3拮抗劑（如Ondansetron）+ NK1拮抗劑
- 前庭性嘔吐：抗組織胺（如Meclizine）更有效

3. 瀉劑使用原則

- 首選：滲透性瀉劑（如Polyethylene glycol）
- 避免長期使用刺激性瀉劑（如Sennosides）

4. 肝膽藥物

- UDCA：需服用數月才能溶解膽固醇結石
- Silymarin：輔助治療酒精性肝病

### 八、感染症治療藥物 (Anti-infective Agents)

藥物分類與特性

1. 抗生素 (Antibiotics)

- β-內醯胺類：青黴素類、頭孢菌素類，干擾細胞壁合成
- 巨環類：如Azithromycin，抑制細菌蛋白質合成
- 四環素類：廣效性，但影響牙齒與骨骼發育
- 喹諾酮類：如Ciprofloxacin，對革蘭氏陰性菌效果佳

2. 抗病毒藥物 (Antivirals)

- 流感治療：如Oseltamivir，抑制神經胺酸酶
- 疱疹治療：如Acyclovir，抑制病毒DNA聚合酶
- HIV治療：多種藥物組合（HAART療法）

3. 抗真菌藥物 (Antifungals)

- 唑類：如Fluconazole，抑制麥角固醇合成
- 多烯類：如Amphotericin B，嚴重全身性感染使用

4. 抗原蟲/寄生蟲藥物

- 瘧疾治療：如Chloroquine（抗藥性嚴重）
- 驅蟲藥：如Mebendazole，治療線蟲感染

臨床注意事項

1. 抗生素使用原則

- 經驗性治療：依感染部位常見菌種選擇
- 降階療法：廣效抗生素後依培養結果調整
- 抗藥性預防：避免不必要使用，完成療程

2. 抗病毒藥物

- 流感：高風險族群優先使用（如老年人、慢性病患者）
- 疱疹：早期治療可縮短病程

3. 抗真菌藥物

- 唑類：監測肝功能（尤其長期使用）
- Amphotericin B：需預先給藥減少輸注反應

4. 特殊感染治療

- MRSA：需使用Vancomycin或Linezolid
- 結核病：多藥合併治療6個月以上

### 九、禿頭與骨質疏鬆藥物 (Hair Loss & Osteoporosis Medications)

藥物分類與特性

1. 禿頭治療藥物

- 雄性禿治療
  - Minoxidil：外用藥，刺激毛囊生長
  - Finasteride：口服藥，抑制DHT（男性專用）
  - Dutasteride：強效5α-還原酶抑制劑
- 其他類型禿頭
  - 類固醇：用於圓形禿（Alopecia areata）
  - 免疫調節劑：如JAK抑制劑（新興療法）

2. 骨質疏鬆治療藥物

- 抗骨吸收藥
  - 雙磷酸鹽類：如Alendronate，抑制破骨細胞
  - 雌激素調節劑：如Raloxifene，選擇性作用
- 促骨生成藥
  - 副甲狀腺素類似物：如Teriparatide，刺激造骨細胞
  - RANKL抑制劑：如Denosumab，減少骨質流失

臨床注意事項

1. 禿頭治療重點

- Minoxidil：
  - 需持續使用4-6個月見效，常見初期脫髮增加
  - 局部可能刺激（改用低濃度或泡沫劑型）
- Finasteride/Dutasteride：
  - 禁忌：孕婦不可接觸（致畸風險）
  - 副作用：性慾降低、精液量減少（通常可逆）
- 女性禿頭：
  - 可考慮低濃度Minoxidil（2%），避免Finasteride

2. 骨質疏鬆治療重點

- 雙磷酸鹽類：
  - 服用方式：早晨空腹，大量水送服，保持直立
  - 風險：罕見下顎骨壞死（ONJ）
- Teriparatide：
  - 療程限制：最長24個月（後需轉用抗吸收藥物）
  - 禁忌：骨癌或放射性骨病變患者避免
- 監測指標：
  - DEXA掃描：每1-2年評估骨密度
  - 鈣/維生素D：需充足補充（尤其Denosumab治療期間）
